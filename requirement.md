Build a modern, conversion-focused UK website inspired by Sunnyplast Romania (https://sunnyplast.ro?lang=en), keeping the core structure but significantly improving UI/UX, clarity, and engagement.

### Goal:

Create a clean, industrial-modern B2B website for a plastic packaging company targeting UK clients. The design should feel premium, fast, and trustworthy.


---

### Data Source Handling:

* Try to analyze and extract structure, content ideas, and product categories directly from the live website (https://sunnyplast.ro?lang=en)
* If any content or structure is unclear, refer to screenshots provided in the /ref folder
* Do NOT blindly copy content — reinterpret and improve it for a UK audience


---

### Design Direction:

* Follow base structure of existing site (Home, Products, About, Contact)
* Do NOT copy design — redesign with modern UI principles
* Use whitespace, strong typography, and grid layouts
* Color palette: neutral industrial tones (white, grey, blue accents)
* Add subtle animations (hover, scroll reveal)

---

### Key Improvements:

1. Clear hero section with:

   * Strong headline (value proposition)
   * CTA: "Request a Quote"
   * Background: product or manufacturing visuals

2. Product Showcase:

   * Card-based layout
   * Categories (e.g., trays, containers, custom packaging)
   * Each product → image + short description + CTA

3. Trust Section:

   * Certifications
   * Clients / industries served
   * Stats (years, production capacity)

4. About Section:

   * Storytelling instead of plain text
   * Manufacturing capabilities

5. Contact / Lead Capture:

   * Sticky CTA
   * RFQ form (bulk inquiry)
   * WhatsApp / quick contact option

---

### Admin Panel (No Backend – Temporary Setup):

* Simple login system (ID/password hardcoded or local auth)
* Admin can:

  * Add/edit/delete products
  * Upload images
  * Update homepage content
* Data storage:

  * Use in-memory database (JSON or SQLite)
  * Structure should be scalable for future backend migration

---

### Tech Constraints:

* Frontend:  Next.js (preferred)
* Styling: Tailwind css 
* Data layer: JSON-based or SQLite (temporary)
* No real backend required now
* Ensure clean separation for future API integration

---

### UX Priorities:

* Mobile-first responsive design
* Fast loading
* Clear navigation
* Conversion-driven layout

---

### Deliverables:

* Home page
* Product listing page
* Product detail page
* About page
* Contact page
* Admin panel (basic CMS-like functionality)

---

### Extra Enhancements:

* Add search/filter in products
* Add “Custom Packaging” highlight section
* Add testimonials or use-case industries
* Add download catalog option (PDF)

---

The final output should feel like a modern UK industrial brand website — not a template clone, but an evolved version of Sunnyplast.
